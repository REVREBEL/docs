> Test Blockquote

The following HTML tags are supported inside MDX. Each tag must be properly closed (e.g. `<br />`, not `<br>`).

```html
<details> & <summary>   // Create collapsible toggles
<sub>                   // Subscripts
<sup>                   // Superscripts
<kbd>                   // Keyboard shortcuts (renders as a little button)
<table>                 // Advanced table formatting beyond Markdown pipes
```
